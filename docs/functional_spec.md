> Imagine a rubber band floor. In a **Positive Gamma** (Sticky) market, the floor pulls price back when it tries to run. In a **Negative Gamma** (Slippery) market, the floor disappears. Dealers must sell *more* as price falls, creating a snowball effect that accelerates the crash.
